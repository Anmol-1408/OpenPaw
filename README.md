# OpenPaw

An AI-powered command-line tool built with Bun and TypeScript, designed for interactive AI-assisted workflows.

## Features
- Interactive AI prompt handling
- Custom terminal UI elements via @clack/core
- OpenRouter AI integration (@openrouter/ai-sdk-provider)
- TypeScript-driven command execution

## Installation
1. Install dependencies:
```bash
bun install
```
2. Requires Bun v1.3.14 or higher

## Usage
```bash
bun run index.ts
```

## Technical Stack
- Bun (fast JavaScript runtime)
- TypeScript 5+
- @clack/core for terminal UI
- @openrouter/ai-sdk-provider for AI capabilities
- ai v6.0.209 for core AI functionality

## Development
- Peer dependencies: TypeScript 5+
